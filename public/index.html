API 接口文档

GET /api/exams

获取所有试卷列表

GET /api/questions?exam_id=ID

获取指定试卷的题目

POST /api/upload

参数: file (docx), uploader (string), subject (string)

POST /api/upload/ai

AI上传（流式返回进度），参数: file (docx), uploader (string), subject (string)

POST /api/upload/json

Body: { uploader?: string, subject?: string, questions: Array } 或直接传数组

POST /api/upload/background

参数: background (图片文件), name (可选, string)

GET /api/backgrounds

参数: page (可选, 默认1), pageSize (可选, 默认20)

POST /api/ask-ai

Body: { question: object, userAnswer?: string }

POST /api/heartbeat

Body: { userId?: string }

GET /api/stats

获取仪表盘数据
